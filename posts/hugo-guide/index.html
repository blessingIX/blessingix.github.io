Hugo Guide

Hugo静态博客搭建

安装hugo

url：https://github.com/gohugoio/hugo/releases

下载安装包：https://github.com/gohugoio/hugo/releases/download/v0.83.1/hugo_0.83.1_Windows-64bit.zip

新建站点

hugo new site <site name>

下载主题

主题地址：https://themes.gohugo.io/

terminal主题为例：

git clone https://github.com/panr/hugo-theme-terminal.git themes/terminal

Github加速源
香港：hub.fastgit.org

使用加速源：

git clone https://hub.fastgit.org/panr/hugo-theme-terminal.git themes/terminal

本地启动

hugo server -t <theme> --buildDrafts

部署到远端（Github）

  1. GitHub上新建一个仓库

    仓库名：$github-username$.github.io

    $github-username$: GitHub用户名

    例如：

    以blessingIX为例，则仓库名为blessingIX.github.io

  2. 关联远端地址

    hugo --theme=$theme$ --baseUrl="https://$github-username$.github.io/" --buildDrafts

    $theme$: 主题名称

    $github-username$: GitHub用户名
